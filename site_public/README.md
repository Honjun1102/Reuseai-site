# Response Cache / Reuse System landing page

This directory is a static GitHub Pages site for the response cache / reuse product.

## Files

- `index.html`: public landing page.
- `styles.css`: site styling.
- `README.md`: deployment notes.

## What to update before publishing

- Update the repo links if you fork or rename the repository.
- If you want email contact, replace the GitHub issue form with your inbox.

## Publish with GitHub Pages

Use the GitHub Actions workflow in `.github/workflows/pages.yml`.
The workflow copies `site_public/` into a Pages artifact and deploys it.

Current public repo:

- https://github.com/Honjun1102/Reuseai-site

Current public landing page:

- https://honjun1102.github.io/Reuseai-site/

## Local preview

Open `site_public/index.html` directly in a browser, or serve the directory with any static file server.
